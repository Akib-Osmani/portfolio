# CV Directory

Place your CV/Resume file here as `Akib_Osmani_CV.pdf` to enable the download functionality.

## Instructions:
1. Create your CV in PDF format
2. Name it exactly: `Akib_Osmani_CV.pdf`
3. Place it in this directory
4. The download buttons will automatically work

## Alternative:
If you prefer a different filename, update the `cvUrl` variable in `assets/js/main.js` to match your file name.
